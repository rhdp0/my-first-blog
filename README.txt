DESAFIO DE ESTÁGIO - JC

Para o projeto foi utilizado:
HTML;
CSS;
JavaScript.

No desafio foi pedido que fosse feito uma página com os seguintes requisitos:
HTML semântico;
Responsividade;
Interações da página em JS.

Com o design fornecido pela empresa no desafio, foi feita essa página com as funcionalidades a seguir:

Header:
A logo redireciona para a home page da empresa.
O mecanismo de pesquisa possibilita obter conteúdo da página "https://ne10.uol.com.br/radar/buscar".
O botão ENTRAR redireciona para a página de login da empresa.

Main:
O usuário poderá filtrar os tópicos com o auxílio do mecanismo "Filtrar" marcando o filtro que deseja aplicar e clicando no 
botão "BUSCAR". Já o botão "LIMPAR BUSCA" limpa os filtros aplicados. Os dois botões chamam uma função em JavaScript específica.

Footer:
O usuário terá acesso as midias sociais e outros meios de entrar em contato com a empresa.
